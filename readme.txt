1,
本项目实现了命令行文本统计程序，能够正确统计导入的纯英文txt文本中的字符数，单词数，句子数。
打开程序会出现如下提示：
wc.exe  [1]   [2]
[1]处输入一个数字，用来选择功能：
“1”表示统计单词数
“2”表示统计字符数
“3”表示统计句子数

[2]处书写文本地址  如：B:\a.txt

如若书写格式错误将会出现“Open the file failure...”
